JavaScript基础
[ JavaScript ]

Posted on 12/13/2022 Symbols count in article: 34k Reading time ≈ 31 mins.

JavaScript基础语法梳理，主要基于 ECMAScript 2015 (ES6) 标准，整理其语言上的数据模型与控制流程，重点关注与其他语言产生差异化的地方，尤其在编码便利性或执行效率上有推动作用的部分。

基本特点

  • 一种多范式（multi-paradigm）动态语言（dynamic language）
    • 支持OOP（object-oriented programming）
      • 对象模型（object model）采用prototype-based，而非class-based
        • 动态继承（dynamic inheritance），属性（property）和方法（method）可以动态的增加到任何对象上
    • 支持函数式编程（functional programming）- functions are First-Class Citizen
  • 借鉴自C、C++和Java，但没有静态类型和强类型检查（strong type checking）
  • 动态数据类型（dynamic typing, loosely typed），无需声明

严格模式（strict mode）

  • Strict mode在ES5时引入，提供了更简洁的版本，但是破坏了现有的代码，也可能降低执行速度
    • 打开方式：'use strict';放到文件或函数的第一行
    • 句法改变（syntactic ）：
      • 禁用with语句：降低执行速度且变量语义更复杂
    • 语义改变（semantics）：
      • 在non-method函数中this值为undefined
    • 更多的错误（error）：
      • 给未声明变量赋值（ReferenceError）
      • 改变只读属性（TypeError）

数据模型（data model）

  • 7种基本类型（primitive type）:
    • 特点：
      • 是immutable，不能被修改
      • 不是object，没有method和property
      • 支持自动转换为包装对象（auto-boxes）
        • "foo".includes("f")隐式创建临时String对象，然后调用String.prototype.includes()
    • 类型：
      • undefined - 值未定义
      • null - 刻意的non-value
      • boolean - true/false
      • number - 整数（位操作时转换为32bit）或浮点数（64bit双精度），eg 42, 3.14159
      • bigint - 大整数，eg 9007199254740992n
      • string - 字符串，eg "hello"
      • symbol - 唯一标识符
  • 关键字（keyword）
    • function: 定义函数
    • function*：定义生成器函数
    • async function：定义异步函数
    • async function*：定义异步生成器函数
    • yield：暂停或继续一个生成器函数
    • yield*：委托给另一个生成器函数或迭代器对象
    • import：导入外部模块中的函数
    • export：导出函数，使得可用于外部模块
    • await：暂停一个异步函数，等待promise的结果
    • new：创建实例
    • delete：删除对象的属性
    • void：计算表达式并丢弃其返回值
    • typedef：探测给定对象的类型
    • class: 定义类
    • super：访问父对象属性
    • this: 与执行上下文有关的一个特殊属性
    • globalThis：全局对象
  • 操作符（operator）
    • 位（bitwise）
      • &：按位与（Bitwise AND）
      • |：按位或（Bitwise OR）
      • ^：按位异或（Bitwise XOR）
      • ~：按位取反（Bitwise NOT）
      • <<：左移
      • >>：右移（按符号位补位）
      • >>>：无符号右移
    • 算术（arithmetic）
      • +true：前缀+表示按数值计算，结果为1（类型自动转换）
      • -'4'：前缀-表示按数值计算后取反，结果为-4（类型自动转换）
      • A++|++A：递增
      • A--|--A：递减
      • 3**2：幂运算，结果为9
      • 3*2：乘运算，结果为6
      • 3/2：除运算，结果为1.5
      • 3+2：加运算，结果为5
      • 3-2：减运算，结果为1
      • 3%2：求余，结果为1
    • 关系（relational）
      • <|<=：小于型
      • >|>=：大于型
      • ==：等于（类型自动转换）
      • ===：严格等于
      • !=：不等于（类型自动转换）
      • !==：严格不等于
      • a instanceof A：判断a是否为A的实例
      • 'year' in car：判断car对象是否有year属性
    • 逻辑（logical）
      • !：逻辑非（Logical NOT）
      • &&：逻辑与（Logical AND）
      • ||：逻辑或（Logical OR）
      • null ?? 'abc'：如果??左边不是null或undefined，就取左边值，否则取右边值
      • (condition ? ifTrue : ifFalse)：三元运算符
  • 表达式（expression）
    • 字面量（literal）:
      • [1, 2, 3]：数组，类似new Array()
      • {1: 'a'}：对象，类似new Object()/Object.create()
      • /ab+c/i：正则表达式，等同new RegExp('ab+c', 'i')
      • `Hi, ${name}`：模板
      • a() * (b() + c())：分组（可应用于函数）
    • 属性访问器（property accessors）：
      • object.property
      • object["property"]
    • 可选链（optional chaining）：属性后带?时，如果其值为undefined或null，返回undefined（不抛异常）
      • obj.val?.prop
      • obj.func?.(args)
    • 构造实例（instance of a constructor）：
      • new Car('red', 1993)：创建一个实例
      • new.target：不通过new操作符调用时此属性值为undefined
    • 导入模块（import module）：
      • await import("module.js")：异步导入
      • import("module.js").then(...).catch(...)：按需导入模块
      • import.meta：将上下文的元数据暴露出来
    • 解构分配（destructuring assignment）
      • [a, b] = arr, { a, b } = obj：允许把数组或对象的属性分配给变量
    • 逗号操作符（Comma operator）
      • ,：允许在一条语句执行多个表达式，然后返回最后一个表达式的结果
    • 传播语法（Spread syntax）
      • ...obj：允许可迭代对象（例如数组）在需要[0~n]个参数地方被展开
  • 内建对象（built-in object）：
    • 特殊值（Value properties）：没有property和method
      • globalThis
      • Infinity
      • NaN
      • undefined
    • 全局函数（Function properties）：不归属某个对象的函数
      • eval()
      • isFinite()
      • isNaN()
      • parseFloat()
      • parseInt()
      • decodeURI()
      • decodeURIComponent()
      • encodeURI()
      • encodeURIComponent()
      • escape() - 弃用
      • unescape() - 弃用
    • 基本对象（Fundamental objects）：补充表示基本的语言结构
      • Object
      • Function
      • Boolean
      • Symbol
    • 错误对象（Error objects）：表示基本错误的对象
      • Error
      • AggregateError
      • EvalError
      • RangeError
      • ReferenceError
      • SyntaxError
      • TypeError
      • URIError
      • InternalError (Non-standard)
    • 数值和日期（Numbers and dates）：表示数值、日期和数学计算的对象
      • Number
      • BigInt
      • Math
      • Date
    • 文本处理（Text processing）：表示处理字符串操作的对象
      • ·String
      • ·RegExp
    • 使用索引的集合（Indexed collections）：表示按index值排序的集合对象
      • Array
      • Int8Array
      • Uint8Array
      • Uint8ClampedArray
      • Int16Array
      • Uint16Array
      • Int32Array
      • Uint32Array
      • BigInt64Array

      • Float32Array
      • Float64Array
    • 使用键的集合（Keyed collections）：表示以key存储的集合对象
      • Map
      • Set
      • WeakMap
      • WeakSet
    • 结构化数据（Structured data）：提供结构化数据缓存和编码的能力
      • ArrayBuffer
      • SharedArrayBuffer
      • DataView
      • Atomics
      • JSON
    • 内存管理（Managing memory）：提供与垃圾回收机制交互的能力
      • WeakRef
      • FinalizationRegistry
    • 控制抽象对象（Control abstraction objects）：适用于异步编程时避免多层回调结构
      • Promise

      • Generator
      • AsyncGenerator
      • AsyncFunction
    • 反射（Reflection）：提供通过实例解析对象结构的能力
      • Reflect
      • Proxy
    • 国际化（Internationalization）：提供国际通用语言之间的字符串比较、数字和日期格式的转换
      • Intl

  • Array.from()将array-like数据转换为Array对象，可代替Array.prototype.map.call()

其他数据结构

Iterable

  • ES6引入的一种遍历数据的新机制：iteration
  • 适用的值：
    • 可迭代：String、Array、Map、Set、DOM-data
    • 不可迭代：普通的字面对象（Plain object, as created by object literals），例如 for (const x of {1: 'a'}) // TypeError
      • 对象的属性（property）默认不可以迭代
  • 基于Iterable访问数据的语言结构：
    • 解构分配（destructuring）：const [a,b] = new Set(['a', 'b', 'c']);
    • for-of循环: for (const x of ['a', 'b', 'c']) { console.log(x); }
    • Array.from()方法: const arr = Array.from(new Set(['a', 'b', 'c']));
    • 传播语法 (…): const arr = [...new Set(['a', 'b', 'c'])];
    • Map和Set的构造器: const set = new Set([‘a’, ‘b’, ‘c’]);`
    • Promise.all()和Promise.race()方法: Promise.all(iterableOverPromises).then(···);
    • 生成器语句(yield*): function* yieldAllValuesOf(iterable) { yield* iterable; }
  • ES6主要的数据结构Array, Map, Set有三个方法会返回iterable对象：
    • entries()
    • keys()
    • values()
  • TypeScriptDefine
  • Symbol.iterator
  • IterateOverProperties
  • CombinatorForIterables

  • let：ES6变量声明（支持Block作用域）
  • const：ES6常量声明，定义时必须初始化，不允许重新赋值（支持Block作用域）

  • BlockScope
  • var & let

  • Function：在函数内部声明的本地（local）变量
  • Block：{}包围的代码（只适用ES6中let/const声明的变量）

函数（function）

  • 闭包（closure）：一种能记住在函数作用域外部定义的变量或函数的能力（不利于垃圾回收）
  • 立即调用函数表达式（IIFE，Immediately-Invoked Function Expression）：函数定义后立即调用一次
    • 可以限定函数内的变量是私有的，外部不能访问
  • ES6支持箭头函数（arrow function）：简短函数的更简洁表示，且对this支持更友好
  • ES6支持函数参数的默认值，且只当参数为undefined时才会触发默认值
  • ES6支持解构分配（destructuring）用于函数参数上，且允许指定默认值，使得代码更清晰简洁
  • ES6支持可变参数，相比ES5的特殊变量arguments的语法更清晰简洁
  • Closure
  • IIFE
  • ArrowFunction
  • DefaultValue
  • DestructuringParameter
  • VaryingParameter

生成器

  • 生成器（generator）为支持协作式（cooperative）编程而设计的函数，支持paused和resumed
  • ES6生成器新语法：
    • function*标识生成器函数或方法，其返回一个生成器对象（generator object）
      • 生成器对象是iterable
    • yield操作符可以暂停生成器，生成器也可以通过它进行输入和输出
    • yield*支持在generator内部调用其他可迭代对象（比如另一个generator）
  • 生成器语法可以简化Promise基础的异步逻辑
  • Basic
  • IterateOverProperties
  • AsynchronousCode
  • yield*
  • Send&RecvValue

类（class）

OOP-Feature

  • 支持可计算的方法名，例如['b'+'ar']: 3 （属性名为bar）
  • 支持生成器方法，例如* [Symbol.iterator]() {...}
  • ObjectLiteral
  • MethodOfObject

Class

  • ES6的class本质还是function，只是提供了更方便的语法形式
  • 类声明不会被提升（hoisted），在执行到它的定义时才会存在
  • 两种定义类的方式：class declarations和class expressions
    • 支持内部名称（lexical inner name），此名称只在类内部实现中可用
  • extends关键字用于创建subclass，且static属性也是被继承的，子类中必须通过super()调用父类构造函数
  • 默认构造器：
  • ClassDeclaration
  • ClassExpression
  • PrivateData
  • PrivateData

模块（module）

  • CommonJS模块：2009年由Kevin Dangoor (Mozilla)发起，通常只用在NodeJS中，默认扩展名.js
    • 导入：require()，synchronous
    • 导出：module.exports
  • ES6模块：2015年引入，可用在Browser和NodeJS（v13.2.0稳定）中，默认扩展名.mjs
    • 导入：import，asynchronous
    • 导出：export
    • 在package.json中增加"type: module"可以启用ES6模块
    • 在browser中通过import()函数返回Promise对象的方式实现动态导入
  • CommonJS

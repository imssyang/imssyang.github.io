Swift基础
[ Swift ]

Posted on 5/13/2024 Symbols count in article: 20k Reading time ≈ 18 mins.

Swift是用于开发IOS和Mac应用的开发语言，从obj-c演变而来，且生态上向前兼容。语言语法有源于cpp的身影，灵活性上相比python更甚，值得看看Apple引领的开发语言演变成的样子。

特点

  • 变量用前初始化（initialized before use）
  • 数组越界检测（out-of-bounds errors），会抛出错误
  • 整型溢出检测（overflow），会抛出错误
  • 通过可选值语义显式处理nil值（optional value）
  • 自动垃圾回收（garbage-collected）
  • 是类型安全的（type-safe），或者说强类型的
  • 支持类型推理（type inference）
  • 支持错误恢复（error recovery）

值（Value）

  • let定义常量，支持运行时初始化（allow unknown at compile-time）
  • var定义变量（like JavaScript）
  • 初始化时支持编译器自动推理值类型

类型（Type）

  • 命名类型（named type）：有具体名称的类型
    • Structure：基本类型在标准库中是基于结构体类型实现的
      • Int: 32/64bit (UInt8, UInt16, Int32, Int64, … like C)
      • Float: 32bit
      • Double: 64bit
      • Bool: true, false
      • String
    • Class
    • Enumeration
    • Protocol
    • 集合类型（collection type）
      • Array
      • Set
      • Dictionary
  • 组合类型（compound type）：没有具体名称的类型
    • Function
    • Tuple: let http404Error = (404, "Not Found") // type (Int, String)
  • 值类型（value type）：赋值和传参按copyed传递（语义同cpp）
    • String
    • Array
    • Dictionary
    • Structure
    • Enumeration
    • Tuple
  • 引用类型（reference type）：赋值和传参按linked传递（语义同cpp）
    • Class
    • Actor
    • Closure
  • 显式类型转换：Int(3.1415)
  • 类型别名：typealias AudioSample = UInt16
  • ValueType
  • ReferenceType

Structure

  • 默认按值传递结构体数据（always copied when passed）

String

  • """str"""定义多行字面量，与尾部双引号所在行相同的缩进字符会被自动移除
  • \()语法支持隐式转换值类型

Class

  • 默认按引用传递数据（passed by reference）
  • class关键字用于创建类，其中定义的变量和函数即为类的属性和方法
  • init()用于定义初始化器（initializer），类似cpp中的构造函数
    • 每个属性都需要赋值，要么在let,var声明中，要么在init()中
    • 属性可以有自定义的getter和setter
    • 在初始化器外部更改属性时，更改前会运行didSet，更改后运行willSet
  • deinit()用于定义析构器（deinitializer），类似cpp中的析构函数
  • override用于标识子类重写父类的实现，否则编译器会抛出error
  • ?.用于访问可选实例的属性
  • Class
  • Subclass

Enumeration

  • 枚举的rawValue属性默认为从0开始的Int类型，也支持浮点数或String类型
  • 枚举允许自定义方法
  • 枚举的case支持携带值，此时枚举项在实例间也是不同的
  • General
  • CaseWithValues

Protocol

  • protocol用于定义协议，类似java中的接口、cpp中的纯虚类
    • mutating标记允许方法修改内部数据
  • class,struct和enum都可以采用协议
  • extension用于为已经存在的类型扩展功能

Error

  • 通过实现Error协议定义错误类型
  • throw在执行流程中抛出error
  • throws标识函数可以抛出error
  • do-catch结构用于捕获error
    • (do) try前置标记后面代码可以抛出error
    • (do) try?前置标记会丢弃错误信息并将结果转为可选的，允许为nil
    • (catch) error为内置错误信息的标识
    • (catch) 支持的匹配模式类似于switch-case
  • defer定义在函数范围前执行的代码，即使抛出了error也会执行它（同golang的语义）
  • General
  • defer

Optional

  • type?标识为optional类型，例如Int和Int?是两种类型，后者允许为nil值
  • optional类型未初始化时，默认为nil值
  • 变量可能会nil值时，也必须是optional的，否则会报compile-time error
  • 相关语法：
    • a?.b?.c传播nil值
    • a ?? b遇到nil值时，使用备选值
    • var!强制解包optional值，遇到nil会触发runtime error
  • type!标识为隐式解包可选值（Implicitly Unwrapped Optional）
    • 用于声明时为nil值，后续一定为非nil值的场景中，每次访问时都无需显式解包
    • 隐式解包时遇到nil值也会触发runtime error

Array

  • [element]定义数组，允许尾部元素后的逗号(,)
  • [index]通过索引访问元素
  • 支持增加元素时自动扩容

Dictionary

  • 字典是未排序（unordered）的集合
  • [key:value]定义字典，允许尾部元素后的逗号(,)
  • [key]通过key访问元素

Tuple

  • (a, b)或(n1: a, n2: b)语法创建tuple，常用于函数返回多个值（类似python的tuple）

Function

  • 函数作为first-class类型，支持作为函数参数和返回值
  • 支持返回多个值（基于tuple）
  • 支持嵌套（nested）函数
  • ->用于分割函数参数和返回值类型
  • 函数参数
    • 前置_限制只能按位置传递参数
    • 前置其他标识限制只能按自定义名称（label）传递参数
  • Argument
  • Return
  • Nested

Closure

  • 闭包（closure）语法是用{}包围的代码块，其中用in将参数和返回值类型与body分隔开
    • 在执行环境作用域中仍能访问被闭包捕获的变量或函数（内嵌函数使用外部变量并作为返回值）
    • 函数实际也是闭包的一种特例
  • $0, $1, ...支持通过数字索引代替参数名字，以创建更简洁的闭包
  • 当函数只有一个闭包类型的参数时，可以省略函数参数上圆括号()

内置函数（Build-in）

assert

precondition

  • 用于越界检查或实参是否有效

控制流（Control Flow）

  • ()语法在条件判断上是可选的
  • {}语法在执行块上是必须的

条件（condition）

if

  • 语法结构中的条件必须是Boolean表达式，不支持将值隐式转为Boolean
  • 支持用在赋值语句（let/var）或返回语句（return）上
  • 用if let语法可以安全的解包optional值
  • General
  • if let

guard

  • guard-else语句在条件为false时执行else代码块（if的变形，提高可读性）

switch

  • 支持任何数据类型和广泛的比较操作
  • 执行完case代码块后自动退出，显式break语句是可选的，使用后会忽略后续的代码块
  • fallthrough语句可以将case转为C-style行为，会继续执行下一个case
  • General
  • fallthrough

循环（loop）

for-in

  • 迭代索引范围（range of indexes）
    • ..<语法支持范围为[from,to)
    • ...语法支持范围为[from,to]
  • break <label>指定终止lable所在的循环
  • continue <label>指定在label所在的循环上继续
  • General
  • Break
  • Continue

while

  • repeat...while语法支持循环体至少运行一次

作用域（scope）

defer

  • defer代码块会延迟到离开作用域时被执行

并发（Concurrency）

  • async定义异步函数
  • async let定义异步值，不会阻塞，支持并行运算（parallel）
  • await阻塞等待异步函数或值
  • Task用于调用异步函数，但不会阻塞当前执行栈
  • actor可看成是异步安全的类，多个异步函数可以同时与同一个actor实例安全的交互
  • async&await
  • Task
  • actor

泛型（Generics）

  • <>标识泛型标识符（同cpp）
  • where后置用于指定泛型的一系列约束条件
    • <T: Equatable> 等同于 <T> ... where T: Equatable
  • General
  • where
